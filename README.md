# このリポジトリについて

Git/GitHub 勉強会の課題用リモートリポジトリです。チーム開発を見据えつつ、個人で練習するための手順をまとめています。

## 進め方
1. このリポジトリをローカルにクローンする。
2. ブランチを切る。
3. ルート直下に自分の苗字ローマ字のディレクトリ（例: `Takami`）を作る。
4. 自分の名前のディレクトリ配下にファイルを作る。（タイトルは任意）
5. その中に 1 問 1 ファイルで問題と回答をまとめる。
6. 取り組むプログラミング問題を選ぶ。（https://kenkoooo.com/atcoder/#/table/ のA問題がおススメ）
7. 問題をコメントアウトで問題文を転記する。
8. 回答を記載する。
9. コミットを行う。コミットは 2 つに分ける。（ファイル作成 + 問題文転記 / 回答転記）
10. ブランチをプッシュする。
11. Pull Request を作成する。（使用問題の URL を記載、レビュー依頼は高見または山嵜）
12. 承認されマージされたら完了。

## 目安
- およそ 1 週間で 1〜3 問を目指してください。
- 不明点は Discord で質問してください。
