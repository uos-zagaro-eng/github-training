# このリポジトリについて

Git/GitHub 勉強会の課題用リモートリポジトリです。チーム開発を見据えつつ、個人で練習するための手順をまとめています。

## 進め方
0. このリポジトリをクローンする。
1. ブランチを作成する。
2. ルート直下に自分の苗字ローマ字のディレクトリ（例: `Takami`）を作る。
3. その中に 1 問 1 ファイルで問題と回答をまとめる（タイトルは任意）。
4. AtCoder A/B 問題または Paiza 練習問題を、コメントアウトで問題文を転記する。
5. 続けて回答を記載する。
6. コミットは 2 つに分ける（ファイル作成 / 問題文転記 / 回答転記）。
7. ブランチをプッシュする。
8. PR を作成する（使用問題の URL を記載、レビュー依頼は高見または山嵜）。
9. 承認されマージされたら完了。

## 目安
- およそ 1 週間で 2〜3 問を目指してください。
- 不明点は Discord で質問してください。
